# 🛒 E-Commerce Sales & Customer Analytics

An end-to-end Data Analytics project that analyzes sales performance, customer behavior, product performance, and business trends using **Python, MySQL, and Power BI**.

This project demonstrates the complete analytics workflow—from understanding raw data and loading it into a relational database to performing advanced SQL analysis and building interactive dashboards for business decision-making.

---

## 📌 Project Overview

The objective of this project is to analyze an e-commerce dataset to uncover valuable business insights related to:

- Sales performance
- Customer purchasing behavior
- Product performance
- Customer segmentation using RFM Analysis
- Business KPIs and strategic insights

The project follows a real-world analytics workflow where data is first explored using Python, stored in MySQL, analyzed using SQL, and finally visualized using Power BI.

---

## 📂 Dataset

**Dataset:** Olist Brazilian E-Commerce Public Dataset

The dataset contains information about:

- Customers
- Orders
- Order Items
- Payments
- Reviews
- Products
- Sellers
- Geolocation
- Product Category Translation
